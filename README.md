# 权限控制 RAG 平台

本项目是基于 RuoYi-Vue 3.2.0 改造的权限控制 RAG 平台，面向大模型向量库场景，提供权限治理、RAG 文件入库、真实向量检索、权限二次过滤、审计留痕、日志导出与行为分析告警能力。

当前项目已完成平台侧权限控制与 RAG 检索治理闭环。系统能够对用户、用户组、权限策略、文档标签和 RAG 检索结果进行统一管理，并在检索过程中完成权限过滤、审计记录和异常行为分析。后续可继续接入 LLM API 或本地大模型，将检索结果作为上下文生成最终自然语言答案。

---

## 一、项目定位

本项目重点解决大模型向量库场景下的权限控制问题。

传统 RAG 系统通常更关注“能否检索到相关内容”，但在企业知识库、涉密文档、分级授权等场景中，还必须解决以下问题：

- 不同用户只能检索自己有权限访问的文档；
- 文档需要绑定知悉范围、密级、用户组等权限标签；
- 向量检索返回结果后仍需要进行二次权限校验；
- 每一次 RAG 检索请求都需要审计留痕；
- 管理员需要查看、导出日志，并分析异常访问行为。

因此，本项目的核心目标是构建一个面向 RAG 检索链路的权限控制平台。

## 三、核心功能说明

### 1. 基础信息管理

系统支持用户、用户组以及用户与用户组关系的管理，为权限上下文生成提供基础数据。

### 2. 权限策略管理

系统支持定义权限策略，包括读取密级、知悉范围、访问对象、访问条件等，用于描述文档访问规则。

### 3. 策略绑定

系统支持将权限策略绑定到用户、用户组、文档或目录等实体上，形成可执行的访问控制关系。

### 4. RAG 文件入库

系统支持通过 RAG 文件入库页面上传文档，并与 RAG Server 联调。

文件入库后，系统会形成三段式存储链路：

- MariaDB：保存文件元数据和权限标签；
- MinIO：保存原始文件对象；
- Milvus：保存文档切块后的向量数据。

### 5. 文档权限标签绑定

系统支持维护文档的权限元数据，包括：

- 文档名称；
- 知悉范围；
- 文档密级；
- 所属用户组；
- 状态；
- 备注信息。

文件入库后，系统会自动生成对应的文档权限标签记录，管理员也可以在页面中进行后期维护。

### 6. 访问请求处理

用户发起 RAG 检索请求后，平台会解析当前用户身份、用户组、权限策略和可访问标签，并构造权限上下文。

### 7. 数据检索

平台侧已完成与 RAG Server 的检索接口联调。平台可调用 RAG Server 的真实检索接口，RAG Server 基于 Milvus 返回候选检索结果。

### 8. 数据过滤模块

RAG Server 返回候选结果后，平台会再次进行权限二次过滤，过滤掉不符合当前用户权限范围的文档结果，防止越权返回。

### 9. 审计日志与日志导出

系统支持 RAG 审计日志记录与导出。

审计日志记录内容包括：

- 用户身份；
- 检索问题；
- 用户组；
- 权限标签；
- metadata filter；
- 原始候选结果；
- 通过过滤的结果；
- 被拦截的结果；
- 最终响应；
- 访问决策；
- 请求耗时。

审计日志页面支持查看 JSON 详情，并支持 Excel 导出。

### 10. 行为分析告警

系统支持基于 RAG 审计日志进行行为分析，并生成行为告警记录。

当前支持的分析类型包括：

- 普通访问记录；
- 权限拒绝访问；
- 二次过滤拦截；
- 异常耗时访问。

管理员可以在 RAG 行为告警页面查看、筛选、删除和导出告警记录。

## 五、启动方式

项目提供了一键启动脚本。

启动系统：

    rag-start

关闭系统：

    rag-stop

系统默认端口：

| 服务 | 端口 |
|---|---|
| 若依前端 | 1024 |
| 若依后端 | 8080 |
| RAG Server | 8081 |
| MinIO | 9000 / 9001 |
| Milvus | 19530 |

启动后访问：

    http://localhost:1024

## 六、演示流程

推荐演示顺序：

1. 首页查看平台功能总览和系统架构图；
2. 查看基础信息、用户组、权限策略和策略绑定；
3. 查看文档权限标签页面，说明文档密级、知悉范围和所属用户组；
4. 进入 RAG 文件入库页面，展示文件上传和入库流程；
5. 进入 RAG 检索页面，展示权限过滤后的检索结果；
6. 进入 RAG 审计日志页面，查看检索审计记录；
7. 点击日志详情，查看 JSON 级别审计链路；
8. 点击日志导出，导出 Excel；
9. 进入 RAG 行为告警页面，点击行为分析；
10. 查看普通访问、异常耗时、拒绝访问等告警记录。
